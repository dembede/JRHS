Summary

Everything in HTML is a box, and follows a box model.

outer display type

Outer display types are two, wither inline or block e.g. p h1, div are block elements by default

The Box Model defines how margins, borders, padding and content work together.

Content box: the place where content is displayed, use inline-size, block-size or width, height

Padding box: just the usual padding applied, use padding

Border box: includes the border that wraps around any element and the padding. Use border to size it

Margin box: just the usual margins applied to elements
