arrays

Topic reference: https://developer.mozilla.org/en-US/docs/Learn/JavaScript/First_steps/Arrays

finding length

Here's how const shopping = ['bread', 'milk', 'cheese', 'hummus', 'noodles'];
console.log(shopping.length); // 5

accessing and modifying

console.log(shopping[0]); // 'bread'

To modify shopping[2] = "eggs" — this changes 'cheese' to 'eggs'

Multidimentional arrays e.g. shopping = [2, 4, 3, 6, 3, [6, 7]] can be accessed like this shopping[2][2]

finding items in arrays

shopping.indexOf('eggs'); // -1 means not found, anything else means found in the said position

adding items to arrays

to add to the begining of the array use unshift() for example:
array.unshift('new-item')

to remove the last item from array use pop(), for example
arr.pop();

to remove the first item of the array use shift()

use splice() if you know the index of the item to be removed. e.g.
arr.splice(where_to_start,how_many_items_to_be_removed)

accessing every item

(1) use for...of to loop through them all. e.g.
for (const item of shopping) { console.log(item) }

Use map() for things you'd like to do repeadedly.e.g. pinting out a list of names

filter() creates new array containing only items that matched the filtering criteria. For example, getting cities with long names:
const cities = ['London', 'Liverpool', 'Totnes', 'Edinburgh'];
const longerThanEight = cities.filter(city => city.length > 8);
// [ "Liverpool", "Edinburgh" ]

converting between strings and arrays

split() and join()

Examples for both
const data = "nairobi,nakuru,naivasha,nyahururu";
const towns = data.split(',');
console.log(towns); // [ "nairobi", "nakuru", "naivasha", "nyahururu" ]
const data2 = towns.join(',');
console.log(data2); // nairobi,nakuru,naivasha,nyahururu

Also, toString() works but is limited to using commas.
towns.toString();
// nairobi,nakuru,naivasha,nyahururu
