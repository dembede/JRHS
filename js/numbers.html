numbers and operators

types of numbers

  • Integers - floating-point numbers, without a fraction. Can be +ve or -ve
  • Floating point numbers (floats) - e.g. 12.345234
  • Doubles - like floating point numbers but with greater precision, more accurate owing to more decimal places

can use typeof method to check what type a variable is holding

useful number methods

NOTE: there's more later to come

toFixed() rounds off to a fixed number of decimal places.

arithmetic operators

There's the common ones like +, -,*, /

then there's the not so common ones like:

  • remainder (or modulo) % - returns the remainder of what's left after dividing a number
    e.g. 8 % 3 = 2
  • exponent ** - raises a base power to the exponent power
    .e.g. 5 ** 2 = 25
