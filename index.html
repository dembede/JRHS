CSS Selectors

CSS selectors define the elements to which a set of CSS rules apply.

Universal selectors

Selects all elements, but can be restricted to specific namespaces e.g.

  • ns|* matches all elements in the "ns" namespace.
  •  |* matches elements without a defined namespace
  • *|* matches all elements

Note on syntax: for simple selectors *.darkmode and .darkmode is really the same thing

  • Element selector : p selects a paragraph tag <p> ... <p>
  • ID selector : #title selects an element with an attribute id="title"
  • Class selector : .title selects an element with an attribute class="title"
  • Attribute selector : targets elements on the page with specific attributes e.g.li[class="light"] This is a span with a class "light"

Attribute selectors

From: https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Selectors

CSS attribute selectors such as a[title] {} (selecting all links with titles) can also be so specific to allow a[href="https://example.com"] { }

Pseudo-classes and pseudo-elements

There's the common pseudo-classes a:hover { } which style different states of an element. But there's the pseudo-elements p::first-line { } or p::first-letter which select a part of the element instead of the entire element itself.

Combinators

Used to target specific elements within the document, for instance p > span { } targets spans within a paragraph
